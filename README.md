# go-tencent-api
Client packages for tencent API, like ExMail, ExWechat, ...


## ExMail

### Prepare

Put Auth string as variable `EXMAIL_API_AUTHS` in Environment

### Features

- `exmail.GetUser(alias string) (*User, error)`
- `exmail.CountNewMail(alias string) (int, error)`
